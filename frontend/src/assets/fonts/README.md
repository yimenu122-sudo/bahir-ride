- Assets Management (Fonts)
- ***
- This folder should contain your font files (e.g., .woff2, .ttf).
-
- Recommended Fonts for Bahir-Ride:
- 1.  Outfit (English/Latin) - Modern, clean, tech-focused.
- 2.  Noto Sans Ethiopic (Amharic) - Best readability for Ge'ez script.
-
- Usage:
- - Place the font files here.
- - Import them in your global CSS (src/index.css) using @font-face.
-
- Example:
- @font-face {
- font-family: 'Outfit';
- src: url('@/assets/fonts/Outfit-Regular.woff2') format('woff2');
- font-weight: 400;
- font-style: normal;
- }
